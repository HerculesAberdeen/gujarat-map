# Gujarat Geothermal Demo Map (Static Leaflet)
This is a ready-to-use static map viewer. No coding or VS Code needed.

## To use
1. Export your QGIS layers as GeoJSON (EPSG:4326 - WGS84).
2. Copy them into the `data/` folder here and name them:
   - gujarat_boundary.geojson
   - gradient_zones.geojson
   - substations_132kv.geojson
   - transmission_lines.geojson
   - cities.geojson
   - hot_springs.geojson
3. Double-click index.html to view locally.
4. To share online, go to vercel.com/new and drag this folder in.
